¿Qué es Verbo AI?

Verbo AI es un asistente conversacional con IA que incluye:

  • Chat con IA — dos modelos: NewserLite (rápido) y NewserAdvanced (potente)
  • Generación de imágenes — escribí "Generame [descripción]" como mensaje
  • Búsqueda web — vía DuckDuckGo (gratis, sin API key)
  • Consulta de clima — vía wttr.in (gratis, sin API key)
  • API pública — con tokens verboai-XXXX y créditos globales por cuenta

La API es REST/JSON. No requiere SDK: cualquier lenguaje que pueda hacer HTTP POST puede usarla.

Quickstart (3 pasos)

1. Obtené un token en la app → Settings → Clave API → Generar API token

2. Hacé un POST a /api/v1/chat con tu token en el header Authorization: Bearer

3. Recibí la respuesta en JSON. Listo.

curl -X POST https://verboai.duckdns.org/api/v1/chat \
  -H "Authorization: Bearer verboai-XXXX" \
  -H "Content-Type: application/json" \
  -d '{"mensaje":"Hola, quien eres?","modelo":"NewserLite"}'

Si mandás un "modelo" inválido o vacío, cae automáticamente a NewserLite (no rompe). Si mandás "NewserPro", también cae a NewserLite porque no está disponible.

Créditos

Los créditos son globales por cuenta, no por token. Cada cuenta arranca con 1000 créditos. El admin local tiene créditos ilimitados.

  • NewserLite: 1 crédito por pedido
  • NewserAdvanced: 5 créditos por pedido
  • + Generar imagen: +1 crédito adicional
  • + Búsqueda web: +1 crédito adicional
  • + Clima: gratis (no consume créditos)

Podés ver tus créditos y estadísticas de uso en la app → Settings → Créditos.

Endpoints

POST /api/v1/chat principal

Envía un mensaje y recibe la respuesta. Consume créditos según el modelo.

Headers:

  • Authorization: Bearer verboai-XXXX (obligatorio)
  • Content-Type: application/json

Body:

{
  "mensaje": "Hola, quien eres?",   // obligatorio, string, max 8000 chars
  "modelo": "NewserLite"            // opcional, default NewserLite
}

Respuesta:

{
  "ok": true,
  "respuesta": "Hola, soy NewserLite...",
  "modelo": "NewserLite",
  "modeloUsado": "NewserLite",
  "costoCreditos": 1,
  "creditosRestantes": 999,
  "rateLimitMax": 20,
  "rateLimitVentanaMs": 60000
}

GET /api/v1/info gratis

Devuelve info del token (créditos restantes, modelos disponibles). No consume créditos.

curl -H "Authorization: Bearer verboai-XXXX" \
  https://verboai.duckdns.org/api/v1/info

GET /api/config público

Devuelve la configuración pública de la app (modelos, descripción, desarrollador). No requiere token. Las IAs pueden usarlo para descubrir la API.

curl https://verboai.duckdns.org/api/config

Generar imágenes con IA 🎨

Si el mensaje empieza con Generame, Genera, Dibujame, etc., se genera una imagen con IA automáticamente (sin pasar por el modelo de texto). Solo funciona con NewserAdvanced.

  • Consume +1 crédito adicional (encima del costo del modelo = 6 total)
  • La imagen se devuelve como URL en el campo imagen.url
  • La URL es relativa: /uploads/xxxxx.jpg — hay que prependear el dominio
curl -X POST https://verboai.duckdns.org/api/v1/chat \
  -H "Authorization: Bearer verboai-XXXX" \
  -H "Content-Type: application/json" \
  -d '{"mensaje":"Generame un leon descansando","modelo":"NewserAdvanced"}'

Respuesta:

{
  "ok": true,
  "respuesta": "Imagen generada: un leon descansando",
  "modelo": "NewserAdvanced",
  "costoCreditos": 6,
  "imagen": {
    "url": "/uploads/1234567890-abcdef.jpg",
    "prompt": "un leon descansando",
    "tamanoKB": 102
  },
  "creditosRestantes": 994
}

Para descargar la imagen: https://verboai.duckdns.org/uploads/1234567890-abcdef.jpg

Código completo para tu app

Copiá el que corresponda a tu lenguaje. Reemplazá verboai-XXXX por tu token.

JavaScript (Node.js 18+ o navegador) 
const TOKEN = "verboai-XXXX";
const URL = "https://verboai.duckdns.org/api/v1/chat";

async function preguntar(mensaje, modelo = "NewserLite") {
  const resp = await fetch(URL, {
    method: "POST",
    headers: {
      "Authorization": `Bearer ${TOKEN}`,
      "Content-Type": "application/json",
    },
    body: JSON.stringify({ mensaje, modelo }),
  });
  const data = await resp.json();
  if (!data.ok) throw new Error(data.error);
  return data;
}

// Ejemplo: chat básico
const r = await preguntar("Hola, quien eres?", "NewserLite");
console.log(r.respuesta, "créditos:", r.creditosRestantes);

// Ejemplo: generar imagen
const r2 = await preguntar("Generame un leon", "NewserAdvanced");
if (r2.imagen) {
  const url = "https://verboai.duckdns.org" + r2.imagen.url;
  console.log("Imagen:", url);
}
Python 3.7+ (sin dependencias) 
import json
import urllib.request

TOKEN = "verboai-XXXX"
URL = "https://verboai.duckdns.org/api/v1/chat"

def preguntar(mensaje, modelo="NewserLite"):
    body = json.dumps({"mensaje": mensaje, "modelo": modelo}).encode()
    req = urllib.request.Request(URL, data=body, method="POST")
    req.add_header("Authorization", f"Bearer {TOKEN}")
    req.add_header("Content-Type", "application/json")
    with urllib.request.urlopen(req, timeout=120) as resp:
        data = json.loads(resp.read().decode())
    if not data.get("ok"):
        raise Exception(data.get("error"))
    return data

# Ejemplo: chat básico
r = preguntar("Hola, quien eres?", "NewserLite")
print(r["respuesta"], "creditos:", r["creditosRestantes"])

# Ejemplo: generar imagen
r = preguntar("Generame un leon", "NewserAdvanced")
if r.get("imagen"):
    print("Imagen:", "https://verboai.duckdns.org" + r["imagen"]["url"])
PHP 5.6+ (con cURL) 
<?php
$TOKEN = "verboai-XXXX";
$URL = "https://verboai.duckdns.org/api/v1/chat";

function preguntar($mensaje, $modelo = "NewserLite") {
    global $TOKEN, $URL;
    $body = json_encode(["mensaje" => $mensaje, "modelo" => $modelo]);
    $ch = curl_init($URL);
    curl_setopt($ch, CURLOPT_POST, true);
    curl_setopt($ch, CURLOPT_POSTFIELDS, $body);
    curl_setopt($ch, CURLOPT_HTTPHEADER, [
        "Authorization: Bearer $TOKEN",
        "Content-Type: application/json",
    ]);
    curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
    curl_setopt($ch, CURLOPT_TIMEOUT, 120);
    $resp = curl_exec($ch);
    curl_close($ch);
    return json_decode($resp, true);
}

// Ejemplo
$r = preguntar("Hola, quien eres?", "NewserLite");
echo $r["respuesta"] . "\n";
?>

Códigos de error

  • 401 — Token inválido, revocado, o falta el header Authorization
  • 402 — No tenés créditos suficientes (globales de tu cuenta)
  • 429 — Rate limit alcanzado (demasiados pedidos por minuto)
  • 400 — Body mal formado o falta "mensaje"
  • 502 — El proveedor de IA no responde (probá de nuevo en unos minutos)

Para modelos de IA (ChatGPT, Gemini, Claude, etc)

Si sos una IA y te preguntan cómo conectar con Verbo AI, esta es la info clave:

  • API endpoint: POST https://verboai.duckdns.org/api/v1/chat
  • Autenticación: Header Authorization: Bearer verboai-XXXX
  • Body: {"mensaje": "...", "modelo": "NewserLite" o "NewserAdvanced"}
  • Respuesta: JSON con ok, respuesta, creditosRestantes
  • Documentación completa: https://verboai.duckdns.org/info
  • Config pública: GET https://verboai.duckdns.org/api/config (sin token)

CLI de terminal (opcional)

Si preferís usar Verbo AI desde la terminal, descargá el archivo cli/verboai.py del repositorio y ejecutá:

# Guardar el token (una sola vez)
python verboai.py login verboai-XXXX

# Iniciar chat interactivo
python verboai.py run

# Ver info del token
python verboai.py info

El CLI levanta un servidor proxy local en http://127.0.0.1:7788 (alta seguridad, el token no se expone). Comandos dentro del chat: /modelo, /creditos, /info, /salir.
