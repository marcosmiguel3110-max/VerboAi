📖 Cómo usar Verbo AI (la web)

Guía rápida para sacarle el jugo a la web, sin tocar código.

  • Chatear: escribí tu mensaje abajo y mandalo. Podés adjuntar hasta 5 imágenes con el clip.
  • Elegir modelo: arriba del chat (o al lado del micrófono) está el selector de modelo. NewserLite es rápido y liviano para el día a día; NewserAdvanced es más potente, genera imágenes, busca en la web y consulta el clima; NewserAdvanced1.5 (nuevo) suma un razonamiento interno aún más profundo y es el mejor para código, ejecutando código real y consultando APIs de prueba.
  • La Biblia completa: desde el menú lateral podés leer la Biblia capítulo por capítulo, marcar tu progreso y volver donde la dejaste.
  • Cuadernito: durante una conversación de estudio bíblico, Verbo AI puede ir armando notas en un panel lateral que podés abrir con el ícono del cuaderno.
  • Nuevas conversaciones: "Nueva conversación" en el menú lateral empieza un chat limpio. Todas tus conversaciones quedan guardadas y las podés buscar, renombrar o borrar desde la lista.
  • Settings: ahí cambiás el modo de respuesta, el fondo/tema, generás tus tokens de API y ves tus créditos.
  • Créditos: cada modelo gasta una cantidad distinta por mensaje (ver "3. Modelos disponibles" más abajo). Se comparten entre la web y tus tokens de API.

1. Obtener un token

Para usar la API necesitás un token tipo verboai-XXXXXXXXXXXX. Para conseguirlo:

  • Entrá a la app e iniciá sesión
  • Andá a Settings → Clave API
  • Hacé clic en "Generar API token"
  • Copialo AHORA: solo se muestra una vez, después solo ves los últimos 4 dígitos

Si no ves la sección "Clave API" en Settings, tu cuenta todavía no tiene acceso (mensaje "Próximamente"). Pedile al admin que te agregue.

2. Endpoints disponibles

POST /api/v1/chat principal

Envía un mensaje y recibe la respuesta. Consume créditos según el modelo.

curl -X POST https://verboai.duckdns.org/api/v1/chat \
  -H "Authorization: Bearer verboai-XXXX" \
  -H "Content-Type: application/json" \
  -d '{
    "mensaje": "Hola, quien eres?",
    "modelo": "NewserLite"
  }'

Respuesta:

{
  "ok": true,
  "respuesta": "Hola, soy NewserLite...",
  "modelo": "NewserLite",
  "modeloUsado": "NewserLite",
  "costoCreditos": 1,
  "creditosRestantes": 999,
  "rateLimitMax": 20,
  "rateLimitVentanaMs": 60000
}

GET /api/v1/info gratis

Devuelve info del token (créditos restantes, modelos disponibles). No consume créditos.

curl -H "Authorization: Bearer verboai-XXXX" \
  https://verboai.duckdns.org/api/v1/info

GET /api/v1/chats gratis nuevo

Devuelve el listado de tus conversaciones guardadas (las mismas que ves en la app web): id, titulo, creadoEn y actualizadoEn. No consume créditos.

curl -H "Authorization: Bearer verboai-XXXX" \
  https://verboai.duckdns.org/api/v1/chats

Respuesta:

{
  "ok": true,
  "total": 2,
  "chats": [
    { "id": "c3b1...", "titulo": "Nueva conversacion", "creadoEn": "2026-07-17T...", "actualizadoEn": "2026-07-17T..." }
  ]
}

GET /api/v1/creditos gratis nuevo

Devuelve solo tus créditos restantes, más liviano que /api/v1/info cuando no necesitás la lista de modelos. No consume créditos.

curl -H "Authorization: Bearer verboai-XXXX" \
  https://verboai.duckdns.org/api/v1/creditos

Respuesta:

{
  "ok": true,
  "creditos": 999,
  "creditosIniciales": 1000,
  "esAdmin": false
}

GET /api/mongo-status admin

Verifica el estado de MongoDB (requiere sesión iniciada).

Si mandás un "modelo" inválido o vacío, cae automáticamente a NewserLite (no rompe).

4. Generar imágenes con IA 🎨

Si el mensaje empieza con Generame, Genera, Dibujame, etc., se genera una imagen con IA automáticamente (sin pasar por el modelo de texto). Solo funciona con NewserAdvanced.

  • Consume +1 crédito adicional (encima del costo del modelo)
  • La imagen se devuelve como URL en el campo imagen.url
  • La URL es relativa: /uploads/xxxxx.jpg — hay que prependear el dominio
curl -X POST https://verboai.duckdns.org/api/v1/chat \
  -H "Authorization: Bearer verboai-XXXX" \
  -H "Content-Type: application/json" \
  -d '{
    "mensaje": "Generame un leon descansando al atardecer",
    "modelo": "NewserAdvanced"
  }'

Respuesta:

{
  "ok": true,
  "respuesta": "Imagen generada: un leon descansando al atardecer",
  "modelo": "NewserAdvanced",
  "costoCreditos": 6,
  "costoBase": 5,
  "costoExtraHerramientas": 1,
  "imagen": {
    "url": "/uploads/1234567890-abcdef.jpg",
    "prompt": "un leon descansando al atardecer",
    "tamanoKB": 102
  },
  "creditosRestantes": 994
}

Para descargar la imagen, prependeá el dominio: https://verboai.duckdns.org/uploads/1234567890-abcdef.jpg

5. Código completo para tu app

Copiá el que corresponda a tu lenguaje. Reemplazá verboai-XXXX por tu token.

JavaScript (Node.js 18+ o navegador) 
const TOKEN = "verboai-XXXX";
const URL = "https://verboai.duckdns.org/api/v1/chat";

async function preguntar(mensaje, modelo = "NewserLite") {
  const resp = await fetch(URL, {
    method: "POST",
    headers: {
      "Authorization": `Bearer ${TOKEN}`,
      "Content-Type": "application/json",
    },
    body: JSON.stringify({ mensaje, modelo }),
  });
  const data = await resp.json();
  if (!data.ok) throw new Error(data.error);
  return data;
}

// Ejemplo: chat básico
const r = await preguntar("Hola, quien eres?", "NewserLite");
console.log(r.respuesta, "créditos:", r.creditosRestantes);

// Ejemplo: generar imagen
const r2 = await preguntar("Generame un leon", "NewserAdvanced");
if (r2.imagen) {
  const url = "https://verboai.duckdns.org" + r2.imagen.url;
  console.log("Imagen:", url);
}
Python 3.7+ (sin dependencias) 
import json
import urllib.request

TOKEN = "verboai-XXXX"
URL = "https://verboai.duckdns.org/api/v1/chat"

def preguntar(mensaje, modelo="NewserLite"):
    body = json.dumps({"mensaje": mensaje, "modelo": modelo}).encode()
    req = urllib.request.Request(URL, data=body, method="POST")
    req.add_header("Authorization", f"Bearer {TOKEN}")
    req.add_header("Content-Type", "application/json")
    with urllib.request.urlopen(req, timeout=120) as resp:
        data = json.loads(resp.read().decode())
    if not data.get("ok"):
        raise Exception(data.get("error"))
    return data

# Ejemplo: chat básico
r = preguntar("Hola, quien eres?", "NewserLite")
print(r["respuesta"], "creditos:", r["creditosRestantes"])

# Ejemplo: generar imagen
r = preguntar("Generame un leon", "NewserAdvanced")
if r.get("imagen"):
    print("Imagen:", "https://verboai.duckdns.org" + r["imagen"]["url"])
PHP 5.6+ (con cURL) 
<?php
$TOKEN = "verboai-XXXX";
$URL = "https://verboai.duckdns.org/api/v1/chat";

function preguntar($mensaje, $modelo = "NewserLite") {
    global $TOKEN, $URL;
    $body = json_encode(["mensaje" => $mensaje, "modelo" => $modelo]);
    $ch = curl_init($URL);
    curl_setopt($ch, CURLOPT_POST, true);
    curl_setopt($ch, CURLOPT_POSTFIELDS, $body);
    curl_setopt($ch, CURLOPT_HTTPHEADER, [
        "Authorization: Bearer $TOKEN",
        "Content-Type: application/json",
    ]);
    curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
    curl_setopt($ch, CURLOPT_TIMEOUT, 120);
    $resp = curl_exec($ch);
    curl_close($ch);
    return json_decode($resp, true);
}

// Ejemplo
$r = preguntar("Hola, quien eres?", "NewserLite");
echo $r["respuesta"] . "\n";
?>

6. Códigos de error

  • 401 — Token inválido, revocado, o falta el header Authorization
  • 402 — No tenés créditos suficientes (ej: NewserAdvanced necesita 5)
  • 429 — Rate limit alcanzado (demasiados pedidos por minuto)
  • 400 — Body mal formado o falta "mensaje"
  • 502 — El proveedor de IA no responde (probá de nuevo en unos minutos)

7. CLI de terminal (opcional)

Si preferís usar Verbo AI desde la terminal, descargá el archivo cli/verboai.py del repositorio y ejecutá:

# Guardar el token (una sola vez)
python verboai.py login verboai-XXXX

# Iniciar chat interactivo
python verboai.py run

# Ver info del token
python verboai.py info

El CLI levanta un servidor proxy local en http://127.0.0.1:7788 (alta seguridad, el token no se expone). Comandos dentro del chat: /modelo, /creditos, /info, /salir.
